A simple plugin for Woocommerce that allows users to scroll through product images on the product archive page.

== Description ==

This plugin will create a gallery of the product images in the product archive page. The images users will scroll through consist of the feature image of the product and the images uploaded for the product gallery.

[youtube http://www.youtube.com/watch?v=Q7dkLR5bik4&list=UUuaTxxBYCTvcQkQbSU-rrTQ]

== Installation ==

1. Download woocommerce-product-archive-image-slider.zip file from the wordpress plugin directory.
2. Upload the zip file via the plugins setting on the Wordpress dashboard.
3. Activate plugin.
